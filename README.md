## About This Repository

This repository is a modern template for TypeScript projects using Jest for testing. It is designed for rapid
development, testing, and continuous integration with the following features:

- **TypeScript-first:** All source and test code is written in TypeScript.
- **Jest Testing:** Pre-configured for unit testing with Jest and ts-jest.
- **ES Module Support:** Uses ES module syntax throughout the codebase and configuration.
- **Ready for Online Editors:** Instantly open and code in Gitpod, GitHub Codespaces, StackBlitz, or Replit.
- **Modern Tooling:** Includes scripts for linting, formatting, and building.
- **Easy Setup:** Just clone, install dependencies, and start coding/testing.

### Quick Start

```sh
npm install
npm test
```

### Folder Structure

- `src/` — TypeScript source files
- `test/` — TypeScript test files
- `jest.config.ts` — Jest configuration in TypeScript
